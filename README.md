# Advice Generator App

A simple React application that fetches and displays quote using the Advice Slip API.

## Introduction
This project is built using React and Axios. It retrieves quote from a public API and displays it to the user. The app also includes a button to fetch new advice on demand.

## Features
- Fetches random advice from the Advice Slip API  
- Displays advice dynamically on the page  
- Button to load new suggestions  
- Built with React class components and Axios  

## Usage
1. Install dependencies using `npm install`.  
2. Run the app with `npm start`.  
3. The app will automatically fetch an advice on load.  
4. Click the "Give Suggestion" button to fetch another advice.
